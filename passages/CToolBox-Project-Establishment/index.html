API

  • sort_bubble(int array[], int total, int order)

    参数1：输入整数数组

    参数2：数组长度

    参数3：排列顺序（1：顺序（从小到大）2：反向顺序（从大到小））其他：原始数组输出

  • sort_select(int array[], int total, int order)

    参数1：输入整数数组

    参数2：数组长度

    参数3：排列顺序（1：顺序（从小到大）2：反向顺序（从大到小））其他：原始数组输出

  • judge_number_perfect(int num)

    输入一个整数并判断它是否是一个完整的数字

    返回值：1：是 0：否
