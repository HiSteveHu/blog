HiSteveHu Website & HEXO Framework Re-Operate Test

Apologize

We have noticed something terrible happened to the our website (HiSteveHu.com).

The specific reason for stopping operations some time ago is due to the launch of the Ministry of Industry and Information Technology of China, which requires all website set in China should be registered. Since we have not done this before, we are forced to stop running.

Solution

In order to ensure the freedom of content and information security of the website, we have no plans to register.

Luckily,we have found excellent solutions to solve the current issues.

  1. We have moved the site from FQIDC to Github Page for hosting.

    GitHub is the world’s leading code hosting platform.We believe it will bring a better access experience to our website.

    It is worth noting that since the GitHub server is not set up in mainland China, Chinese users may have stability and speed problem when visiting websites. However, due to the adoption of CDN acceleration services by Github, these shortcomings have been largely resolved.

    The good news is that the re-operated new website uses some new technologies, which will be detailed below.

  2. We have refactored the whole website.

    We define the new site as a platform that focuses on sharing information technology experiences and publishing some interesting things and a place for free thoughts.

    Therefore, this means that we may have deleted other wonderful content from the original website, we apologize for this. However we believe that a more focused direction can lead to more high quality content.

One More Thing

As mentioned above, we have used the following technologies in our re-run new website to improve overall performance and experience:

  • Github Hosting/Page Service

    Stable, popular, safe.

  • HTTPS Encrypted Connection

    Improve connection security

  • CDN Acceleration

    Increase access speed, especially for Chinese users

  • HEXO Framework

    A fast, concise, and efficient framework that replaces the original way of creating web pages, allowing us to focus more on high-quality content.

  • Polar Bear Theme

    A simple set of theme templates

  • New Website Icon

    ……

We hope you will like this new change and add this website to your favorites.

Thanks !
